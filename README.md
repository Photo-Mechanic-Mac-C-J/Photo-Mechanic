## Overview

<a href="#">Photo Mechanic</a> delivers professional-grade photo management through an interface designed for both simplicity and depth. The application serves photojournalists, sports photographers, wedding photographers, and anyone who needs to process large image volumes quickly.

The <a href="#">contact sheet printing</a> produces proof sheets for client review. Generate printed or PDF contact sheets with customizable layouts.

<a href="#">FTP upload</a> transmits images directly to servers during events. Send selected photos to newsrooms or agencies without leaving the application.

<a href="#">Code replacement</a> expands abbreviations into full metadata entries. Type short codes that automatically expand to complete captions or keywords.

The <a href="#">structured keywords</a> organize vocabulary hierarchically for consistent application. Build keyword libraries that ensure metadata consistency across projects.

<a href="#">GPS support</a> displays and manages location information embedded in images. View where photos were taken and add location data when missing.

<a href="#">Photo Mechanic Plus</a> adds cataloging capabilities for library management. Build searchable databases of images across multiple shoots and years.

<a href="#">RAW+JPEG handling</a> manages paired files intelligently. Work with both files as single items or separately as workflow requires.

<a href="#">Quick preview</a> shows selected images larger instantly. Examine detail without opening separate windows or applications.

<a href="#">Filter bar</a> narrows displayed images based on criteria. Show only rated images, specific dates, or other filtered subsets.

<a href="#">External editor integration</a> sends images to Lightroom, Photoshop, or other applications. Launch editing applications with selected images ready to work.
